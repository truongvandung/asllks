Dynamic FormView Advance Odoo

Change The Odoo Form view On the fly without any technical knowledge

Link Video Demo: https://youtu.be/8RCVtkbY6qo

Key Features

  • Change Attribute with Condition
  • Can change field in form tree
  • You can change required/readonly/invisible with state on form
  • Change Required of field
  • Change Readonly of field
  • Change Invisible of field
  • Show/Hide More Field on Form
  • Change String of Columns
  • Can apply for all User
  • Settings per users
  • Form view remains changed on page refresh
  • Applies on all the FormView of Odoo
